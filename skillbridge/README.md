# SkillBridge

A responsive student skill showcase website built with HTML5, CSS3, JavaScript, and Bootstrap 5.

## Pages included
- Home
- About
- Skills
- Projects
- Achievements
- Internships
- Resources
- Community
- Blog
- Contact
- Login
- Register
- Dashboard

## Features
- Sticky navigation
- Responsive layout
- Blue and white professional color theme
- Font Awesome icons and Google Fonts
- Smooth scrolling and hover animations
- Separate CSS and JavaScript files

## Run locally
Open index.html in your browser, or serve the folder with a simple local server such as:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000/
